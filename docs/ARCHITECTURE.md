# 架构设计

## 模块

1. **前端 Web Studio**：React + Vite，本地浏览器运行。
2. **后端 API**：FastAPI，提供项目、Agent、模型、评审、视频任务接口。
3. **SQLite 数据库**：本地持久化项目、产物、模型配置和视频任务。
4. **LLM Provider 层**：统一 OpenAI-Compatible `/chat/completions` 协议，支持 OpenAI、xAI/Grok、SiliconFlow、DeepSeek、OpenRouter、自定义 API。
5. **Agent 工作流引擎**：按短剧工业流程串联 13 个角色。
6. **多模型交叉评审**：多个模型独立评分，系统聚合平均分和修改建议。
7. **视频任务系统**：把分镜和视频 Prompt 转为任务，后续接 Kling、Runway、Veo、海螺、Seedance 等视频 API。

## 数据流

```text
用户创意
  ↓
项目圣经 / 市场雷达 / 爆款拆解
  ↓
脑洞策划 / 人设 / 分集大纲
  ↓
剧本 / 台词嘴替 / 分镜
  ↓
视频 Prompt / 美术一致性 / 合规审核
  ↓
视频任务队列
  ↓
数据复盘与下一轮创作
```

## 角色职责

- 总导演：统一风格、卖点、边界。
- 市场雷达：平台和受众判断。
- 爆款拆解：提炼可复用套路。
- 脑洞策划：强化设定和反转。
- 人设专员：设计人物欲望、秘密、关系冲突。
- 分集大纲：集集钩子、集集尾钩。
- 编剧：写可拍分场剧本。
- 台词嘴替：把台词改短剧化、口播化、可传播。
- 分镜导演：镜头、景别、构图、运镜。
- 视频提示词：生成中英双语视频 Prompt。
- 美术资产一致性：角色脸、服装、场景、道具一致。
- 合规审核：风险检查和替代表达。
- 数据复盘：根据完播、互动、转化迭代剧情。
